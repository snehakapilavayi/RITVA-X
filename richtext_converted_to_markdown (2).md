\# 📄 RITVA X — Software Requirements Specification (SRS)

\## 🧠 Overview

This document defines the requirements for RITVA X built using React (Vite) and Node.js.

\---

\## 🎯 Functional Requirements

\### Task Management

\- Create tasks

\- Edit tasks

\- Delete tasks

\- Categorize (To-Do, Scheduled, Completed)

\---

\### Planner

\- Drag & drop tasks

\- Assign time slots

\---

\### Alarm System

\- Trigger at task time

\- Show decision popup

\---

\### Decision System

\- Execute (YES)

\- Avoid (NO)

\- Allow rescheduling

\---

\### Notes

\- Write notes

\- Save notes

\- Locked journal support

\---

\## ⚙️ Non-Functional Requirements

\- Fast performance

\- Responsive UI

\- Minimal design

\- Secure note storage

\---

\## 🏗️ System Environment

\- Frontend: React (Vite)

\- Backend: Node.js (Express)

\---

\## 🔁 System Flow

1\. Add task

2\. Schedule task

3\. Alarm triggers

4\. Decision

5\. Reflection

\---

\## ⚠️ Constraints

\- Internet required for API features

\- Personal-use system

\---

\## 🚀 Future Enhancements

\- AI tracking

\- Analytics

\- Notifications
